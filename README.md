# Gwadar Ambulance Service Backend

This is the backend server for the Gwadar Ambulance Service System. It is built using Node.js, Express, and Socket.io for real-time ambulance tracking.

## Deployment on Hugging Face Spaces

This repository is pre-configured to deploy the backend on Hugging Face Spaces using the Docker SDK.
